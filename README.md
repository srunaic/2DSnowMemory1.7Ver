# 2D RPG Adventure & Story Game
코마 환자의 정신세계를 표현한 게임 

## 🖥️ 프로젝트 소개
코마 환자의 혼란한 정신 속 세계를 스토리에 담아 표현한 게임
이 환자의 정신속 세계를 돌아다니면서 주인공의 정체성을 찾는 게임
